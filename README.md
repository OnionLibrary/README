# How We Work
1. 요구사항 분석
    - 고객의 요구사항을 분석하여 기능을 명세합니다.
    - 명세된 기능을 정리하고 고객 Confirm시 작업에 착수합니다.
2. 설계
    - 명세된 기능을 기반하여 System 설계를 진행합니다.
    - 미팅을 통해 구현 방향성 제시하고 System 전반적인 개요를 설명해 드립니다.
3. 구현
    - 설계서 기반 구현을 진행합니다.
5. 피드백
    - 최종 SW(HW) 시연 영상을 통해 기능 검증
